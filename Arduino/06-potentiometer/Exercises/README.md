## English (EN)

1. **Two LEDs controlled by the potentiometer**
   Add **one more LED** and modify the program so that **both LEDs are controlled by the potentiometer**.

2. **Inverted LED behavior**
   Modify the program so that the LEDs work in an **inverted way**: as the **potentiometer changes**, one LED **turns on** while the other **turns off**.

   Add **one more LED**. As the **potentiometer changes**, the LEDs should **turn on progressively** (for example: 1 LED, then 2, then 3).

   Turn the **potentiometer reading** into a **function** (for example: `int readPotentiometer()`).

6. **Function to control the LEDs**
   Turn the **LED control logic** into a **function** that receives the processed value and decides which LEDs should be on.

7. **Function to control the buzzer**
   Turn the **buzzer control** into a **function** that receives the value and decides when the buzzer should beep.
