# 🔋 Arduino Projects and Exercises

## 📖 About / Sobre

**EN:**
This directory contains Arduino projects and exercises developed for **learning and practice**, organized by topics and lessons.
All activities are based on **academic classes from Fatec Rubens Lara**, as part of the *Analysis and Systems Development (ADS)* program.

The goal of this repository is to document practical experiments with electronics and programming using Arduino, combining theory and hands-on implementation.

**PT-BR:**
Este diretório contém projetos e exercícios de Arduino desenvolvidos para **aprendizado e prática**, organizados por tópicos e aulas.
Todas as atividades são baseadas nas **aulas da Fatec Rubens Lara**, no curso de *Análise e Desenvolvimento de Sistemas (ADS)*.

O objetivo deste repositório é documentar experimentos práticos com eletrônica e programação utilizando Arduino, unindo teoria e prática.

## 📂 Structure / Estrutura

Each topic follows a standardized structure to make navigation easier:

Cada conteúdo segue uma estrutura padronizada para facilitar a navegação:

```
topic-name/
├── exercise/
│ ├── n-exercise.ino
│ └── README.md
├── circuit.png
├── scheme.pdf
└── README.md
```

### 📌 Description / Descrição

**EN:**

* `exercicios/` → Source code for the exercises
* `circuit.png` → Visual representation of the assembled circuit
* `scheme.pdf` → Circuit schematic diagram
* `README.md` → Description of the exercises and **Tinkercad simulation link**

**PT-BR:**

* `exercicios/` → Código-fonte dos exercícios
* `circuit.png` → Representação visual do circuito montado
* `scheme.pdf` → Esquema elétrico do circuito
* `README.md` → Descrição dos exercícios e **link da simulação no Tinkercad**

## 🧪 Learning Approach / Abordagem de Aprendizado

**EN:**
Each project focuses on reinforcing key concepts such as:

* Basic electronics (LEDs, resistors, sensors)
* Digital and analog inputs/outputs
* Logic building with Arduino code
* Circuit design and simulation

**PT-BR:**
Cada projeto tem como foco reforçar conceitos importantes como:

* Eletrônica básica (LEDs, resistores, sensores)
* Entradas e saídas digitais e analógicas
* Construção de lógica com código Arduino
* Montagem e simulação de circuitos
