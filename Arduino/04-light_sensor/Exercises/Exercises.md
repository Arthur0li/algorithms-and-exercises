## English (EN)

1. **Buzzer + LED**
   Add a **buzzer** to the circuit. Modify the program so that when the **LED turns on**, the **buzzer also beeps**.

2. **Progressive LEDs according to light level**
   Add **two more LEDs** to the circuit. As the **light level decreases**, the LEDs should **turn on progressively** (for example: 1 LED, then 2, then 3).
   The **buzzer must beep when the last LED turns on**.

3. **Function for sensor reading**
   Turn the **light sensor reading** into a **function** that returns the measured value (for example: `int readLightSensor()`).

4. **Function to control the LEDs**
   Turn the **LED control logic** into a **function** that receives the **sensor value** as a parameter and decides which LEDs should be on.

5. **Function to control the buzzer**
   Turn the **buzzer control** into a **function** that receives the **sensor value** and decides when the buzzer should beep.
