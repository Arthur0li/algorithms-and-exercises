# ☕ Java Exercises

## 📖 About / Sobre

**EN:**
This directory contains Java programming exercises developed for **practice and learning**, organized by topics and concepts.
All exercises are based on **academic classes from Fatec Rubens Lara**, as part of the *Analysis and Systems Development (ADS)* program.

The repository focuses on building a solid foundation in Java, with emphasis on object-oriented programming, code organization, and problem-solving.

**PT-BR:**
Este diretório contém exercícios de programação em Java desenvolvidos para **prática e aprendizado**, organizados por tópicos e conceitos.
Todos os exercícios são baseados nas **aulas da Fatec Rubens Lara**, no curso de *Análise e Desenvolvimento de Sistemas (ADS)*.

O foco deste repositório é construir uma base sólida em Java, com ênfase em programação orientada a objetos, organização de código e resolução de problemas.

## 🧠 Learning Approach / Abordagem de Aprendizado

**EN:**
The exercises are designed to reinforce key programming concepts, including:

* Variables and data types
* Operators (arithmetic, logical, relational)
* Conditional structures (`if`, `else`, `switch`)
* Repetition structures (`for`, `while`, `do while`)
* Methods and code organization
* Introduction to Object-Oriented Programming (OOP)
* Problem-solving and algorithm design

As the repository evolves, more advanced Java concepts may be included.

**PT-BR:**
Os exercícios são voltados para reforçar conceitos fundamentais de programação, incluindo:

* Variáveis e tipos de dados
* Operadores (aritméticos, lógicos e relacionais)
* Estruturas condicionais (`if`, `else`, `switch`)
* Estruturas de repetição (`for`, `while`, `do while`)
* Métodos e organização de código
* Introdução à Programação Orientada a Objetos (POO)
* Resolução de problemas e construção de algoritmos

Com o avanço dos estudos, conceitos mais avançados de Java podem ser incluídos.
