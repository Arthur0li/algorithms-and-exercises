# 📚 Algorithms and Exercises

## 📖 About

This repository gathers **programming exercises and algorithm practice**, organized by language and topic. It serves as a personal study space to improve problem-solving skills and strengthen programming fundamentals.

The studies in this repository are based on **books, academic classes, and online courses**, reflecting my continuous learning journey in different programming languages.

## 📂 Structure

* `python/` → exercises in Python
* `c/` → exercises in C
* `java/` → exercises in Java
* `arduino/` → exercises and projects using Arduino

Each exercise folder may contain a `README.md` describing the task, context, and solution.

## 📝 Notes

* This repository is focused on **practice and learning**, so exercises may vary in complexity.
* **Credits and references** (such as books, courses, or professors) are provided inside each exercise or folder whenever applicable.

## 🎯 Goals

* Strengthen logic and algorithmic thinking
* Practice syntax across multiple languages
* Build consistency through regular problem-solving
* Track learning progress over time
