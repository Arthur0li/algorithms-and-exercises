# 🧩 Funções e Operações Aritméticas / Functions and Arithmetic Operations

## Português (PT-BR)

1. **Análise de dois números**
   Escreva um programa que receba **dois números inteiros** e execute diferentes operações com base nas condições:

   * ➕ Se **ambos forem positivos**, calcule a **soma de todos os números entre eles**
   * ✖️ Se **ambos forem negativos**, calcule a **multiplicação de todos os números entre eles**
   * ➗ Se o **primeiro for positivo** e o **segundo negativo**, realize a **divisão entre eles**

2. **Fatorial (N!)**
   Escreva um programa que receba um número **N**.
   Crie uma função que calcule o **fatorial de N (N!)** e retorne o resultado.
   O valor deve ser exibido na função **`main()`**.

   **Exemplo:**
   N = `4`
   Resultado = `24`

3. **Operações aritméticas com funções**
   Escreva um programa que receba dois números inteiros na função **`main()`**.
   Crie funções separadas para cada operação:

   * soma
   * subtração
   * multiplicação
   * divisão

   Utilize **`return`** para retornar os resultados e exiba todos na função `main()`.

4. **Calculadora com switch e funções**
   Escreva um programa semelhante ao anterior, mas utilizando o comando **`switch`**.
   As operações devem ser implementadas como **funções**, e o **resultado acumulado** deve ser exibido na `main()`.

## English (EN)

1. **Two-number analysis**
   Write a program that receives **two integers** and performs different operations based on conditions:

   * ➕ If **both are positive**, calculate the **sum of all numbers between them**
   * ✖️ If **both are negative**, calculate the **product of all numbers between them**
   * ➗ If the **first is positive** and the **second is negative**, perform the **division between them**

2. **Factorial (N!)**
   Write a program that reads a number **N**.
   Create a function that calculates the **factorial of N (N!)** and returns the result.
   The result must be displayed in the **`main()`** function.

   **Example:**
   N = `4`
   Result = `24`

3. **Arithmetic operations with functions**
   Write a program that receives two integers in the **`main()`** function.
   Create separate functions for each operation:

   * addition
   * subtraction
   * multiplication
   * division

   Use **`return`** to return the results and display all outputs in `main()`.

4. **Calculator using switch and functions**
   Write a program similar to the previous one, but using the **`switch`** statement.
   The operations must be implemented as **functions**, and the **accumulated result** should be displayed in `main()`.
