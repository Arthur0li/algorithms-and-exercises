# 📊 Vetores e Matrizes / Arrays and Matrices

## Português (PT-BR)

1. **Separação por posição (índice par e ímpar)**
   Escreva um programa que receba um **vetor de inteiros com 10 posições** e o separe em dois novos vetores:

   * um contendo os elementos de **índices pares**
   * outro contendo os elementos de **índices ímpares**

   **Exemplo:**
   Vetor original:
   `v = {3, 5, 6, 8, 1, 4, 2, 3, 7, 9}`

   Vetores gerados:
   `pares = {3, 6, 1, 2, 7}`
   `impares = {5, 8, 4, 3, 9}`

2. **Soma de matrizes 2x3**
   Declare duas **matrizes inteiras 2x3**.
   O programa deve:

   * ler os valores das duas matrizes via teclado
   * somar os elementos de **mesma posição**
   * armazenar o resultado em uma **terceira matriz**

   Ao final, exiba:

   * o **índice (linha e coluna)**
   * os **valores correspondentes das duas matrizes**
   * o **resultado da soma**

3. **Ordenação em ordem crescente**
   Escreva um programa que receba **10 números inteiros** via teclado.
   Os valores devem ser **ordenados em ordem crescente durante a inserção**.
   Ao final, exiba o vetor com todos os valores **ordenados**.

## English (EN)

1. **Separation by position (even and odd indices)**
   Write a program that reads an **integer array of size 10** and splits it into two arrays:

   * one containing elements at **even indices**
   * another containing elements at **odd indices**

   **Example:**
   Original array:
   `v = {3, 5, 6, 8, 1, 4, 2, 3, 7, 9}`

   Generated arrays:
   `even = {3, 6, 1, 2, 7}`
   `odd = {5, 8, 4, 3, 9}`

2. **2x3 matrix sum**
   Declare two **2x3 integer matrices**.
   The program must:

   * read values for both matrices from input
   * add elements at the **same position**
   * store the result in a **third matrix**

   Then display:

   * the **index (row and column)**
   * the **values from both matrices**
   * the **sum result**

3. **Ascending order sorting**
   Write a program that reads **10 integers** from input.
   The values must be **sorted in ascending order as they are entered**.
   At the end, display the array with all values **sorted**.
