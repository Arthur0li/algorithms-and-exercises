# 🗂️ Estruturas de Dados (Structs) / Data Structures (Structs)

## Português (PT-BR)

1. **Cadastro com vetor de estruturas**
   Escreva um programa que utilize um **vetor de estruturas com 4 elementos**.
   Os dados devem ser:

   * recebidos em uma **função**
   * exibidos em outra **função**

   Implemente um **menu interativo** com opção para **encerrar o programa**.

   **Estrutura:**

   * nome
   * endereço
   * cidade
   * estado
   * CEP

2. **Diferença de dias entre duas datas**
   Escreva um programa que receba **duas datas** na função `main()`.
   Crie uma função que calcule o **número de dias entre essas datas**
   e exiba o resultado na `main()`.

   Utilize **vetor de estruturas**.

   **Estrutura:**

   * dia
   * mês
   * ano

3. **Cadastro e cálculo de IMC**
   Escreva um programa que utilize um **vetor de estruturas com 4 elementos**.
   O programa deve permitir:

   * cadastro dos dados
   * exibição dos dados
   * cálculo do **IMC** de todas as pessoas

   Implemente um **menu com as opções**:

   1. receber todos os dados
   2. imprimir todos os dados
   3. calcular IMC
   4. sair

   **Fórmula do IMC:**
   `IMC = peso / (altura * altura)`

   **Estrutura:**

   * nome
   * peso
   * altura

## English (EN)

1. **Struct array registration system**
   Write a program that uses an **array of structs with 4 elements**.
   The data must be:

   * read in one **function**
   * displayed in another **function**

   Implement an **interactive menu** with an option to **exit the program**.

   **Structure:**

   * name
   * address
   * city
   * state
   * ZIP code

2. **Difference in days between two dates**
   Write a program that reads **two dates** in the `main()` function.
   Create a function that calculates the **number of days between these dates**
   and displays the result in `main()`.

   Use an **array of structs**.

   **Structure:**

   * day
   * month
   * year

3. **Registration and BMI calculation**
   Write a program that uses an **array of structs with 4 elements**.
   The program must allow:

   * data input
   * data display
   * calculation of **BMI** for all registered people

   Implement a **menu with the following options**:

   1. input all data
   2. display all data
   3. calculate BMI
   4. exit

   **BMI Formula:**
   `BMI = weight / (height * height)`

   **Structure:**

   * name
   * weight
   * height
