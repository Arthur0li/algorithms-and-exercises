# 💻 C Exercises

## 📖 About / Sobre

**EN:**
This directory contains C programming exercises developed for **practice and learning**, organized by topics and concepts.
All exercises are based on **academic classes from Fatec Rubens Lara**, as part of the *Analysis and Systems Development (ADS)* program.

The repository focuses on building a strong foundation in C, emphasizing logic, memory understanding, and structured programming.

**PT-BR:**
Este diretório contém exercícios de programação em C desenvolvidos para **prática e aprendizado**, organizados por tópicos e conceitos.
Todos os exercícios são baseados nas **aulas da Fatec Rubens Lara**, no curso de *Análise e Desenvolvimento de Sistemas (ADS)*.

O foco deste repositório é construir uma base sólida em C, com ênfase em lógica, compreensão de memória e programação estruturada.

## 🧠 Learning Approach / Abordagem de Aprendizado

**EN:**
The exercises are designed to reinforce key programming concepts, including:

* Variables and data types
* Operators (arithmetic, logical, relational)
* Conditional structures (`if`, `else`, `switch`)
* Repetition structures (`for`, `while`, `do while`)
* Basic input/output
* Problem-solving and algorithm design
* Pointers, Data Structures, File Manipulation

As the repository evolves, more advanced topics may be included.

**PT-BR:**
Os exercícios são voltados para reforçar conceitos fundamentais de programação, incluindo:

* Variáveis e tipos de dados
* Operadores (aritméticos, lógicos e relacionais)
* Estruturas condicionais (`if`, `else`, `switch`)
* Estruturas de repetição (`for`, `while`, `do while`)
* Entrada e saída de dados
* Resolução de problemas e construção de algoritmos
* Ponteiros, Estrutura de Dados, Manipulação de Arquivos

Com o avanço dos estudos, tópicos mais avançados podem ser incluídos.
