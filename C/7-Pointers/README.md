# 👉 Ponteiros / Pointers

## Português (PT-BR)

1. **Média com ponteiros**
   Escreva um programa que receba **números inteiros positivos** via teclado utilizando **ponteiros**.
   Quando um **número negativo** for digitado, o programa deve parar e calcular a
   **média dos valores informados**, exibindo o resultado na tela.

   ⚠️ Regra:

   * todos os cálculos devem ser feitos **utilizando ponteiros**

2. **Comparação de strings com ponteiros**
   Escreva um programa que receba **duas strings de até 10 caracteres** na função `main()`.
   Crie uma função que compare as strings **utilizando ponteiros** e retorne:

   * `1` → se forem **iguais**
   * `0` → se forem **diferentes**

   O resultado deve ser exibido na função `main()`.

   ⚠️ Regras:

   * os vetores devem ser declarados como **variáveis globais**
   * a comparação deve ser feita **utilizando ponteiros**

3. **Busca de letra em vetor (com ponteiros)**
   Escreva um programa que receba **uma letra via teclado utilizando ponteiro**.
   Crie uma função que verifique se essa letra **existe no vetor** abaixo, utilizando
   **ponteiros**, e retorne o resultado.

   O resultado deve ser exibido na função `main()`.

   ⚠️ Regras:

   * o vetor deve ser declarado como **variável global**
   * a letra deve ser passada como parâmetro para a função

   **Vetor:**
   `{ b, d, f, h, j, k, m, o, q, s, u, w, y }`

4. **Structs com ponteiros**
   Escreva um programa que utilize um **vetor de estruturas com 4 elementos**.
   Os dados devem ser:

   * recebidos em uma **função utilizando ponteiros**
   * exibidos em outra **função utilizando ponteiros**

   Utilize um **laço de repetição** para percorrer o vetor.

   ⚠️ Regra:

   * o vetor de estruturas deve ser declarado como **variável global**

   **Estrutura:**

   * nome
   * endereço
   * cidade
   * estado
   * CEP

5. **Datas com ponteiros**
   Escreva um programa que utilize a estrutura abaixo.
   Receba **duas datas via teclado utilizando ponteiros**.
   Crie uma função que calcule o **número de dias entre as datas**, também utilizando ponteiros,
   e exiba o resultado na tela.

   Utilize um **vetor de estruturas**.

   **Estrutura:**

   * dia
   * mês
   * ano

## English (EN)

1. **Average using pointers**
   Write a program that reads **positive integers** using **pointers**.
   When a **negative number** is entered, the program should stop and calculate the
   **average of the entered values**, displaying the result.

   ⚠️ Rule:

   * all calculations must be done **using pointers**

2. **String comparison using pointers**
   Write a program that reads **two strings of up to 10 characters** in `main()`.
   Create a function to compare them **using pointers**, returning:

   * `1` → if they are **equal**
   * `0` → if they are **different**

   Display the result in `main()`.

   ⚠️ Rules:

   * arrays must be declared as **global variables**
   * comparison must be done **using pointers**

3. **Character search in array (with pointers)**
   Write a program that reads a **character using a pointer**.
   Create a function that checks whether this character **exists in the array** below,
   using **pointers**, and returns the result.

   Display the result in `main()`.

   ⚠️ Rules:

   * the array must be declared as a **global variable**
   * the character must be passed as a parameter

   **Array:**
   `{ b, d, f, h, j, k, m, o, q, s, u, w, y }`

4. **Structs with pointers**
   Write a program that uses an **array of structs with 4 elements**.
   The data must be:

   * read in one **function using pointers**
   * displayed in another **function using pointers**

   Use a **loop** to iterate through the array.

   ⚠️ Rule:

   * the struct array must be declared as a **global variable**

   **Structure:**

   * name
   * address
   * city
   * state
   * ZIP code

5. **Dates using pointers**
   Write a program using the structure below.
   Read **two dates using pointers**.
   Create a function that calculates the **number of days between the dates**, also using pointers,
   and display the result.

   Use an **array of structs**.

   **Structure:**

   * day
   * month
   * year
