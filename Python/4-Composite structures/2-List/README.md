# 🐍 Python List Exercises

## 🇧🇷 Português (PT-BR)

1. Faça um algoritmo que permita ao usuário inserir vários números, armazenando-os em uma lista. Ao final, exiba os números em ordem crescente e decrescente.

2. Faça um algoritmo que permita ao usuário inserir vários números e armazená-los em uma lista. Quando o usuário decidir parar, exiba a soma de todos os números armazenados.

3. Faça um algoritmo que simule um sistema de registro de produtos.

   O usuário deve poder:

   * Adicionar produtos com seus respectivos preços
   * Modificar informações de um produto
   * Excluir um produto
   * Atualizar os dados de um produto

## 🇺🇸 English (EN)

1. Write an algorithm that allows the user to enter multiple numbers, storing them in a list. At the end, display the numbers in ascending and descending order.

2. Write an algorithm that allows the user to enter multiple numbers and store them in a list. When the user decides to stop, display the sum of all stored numbers.

3. Write an algorithm that simulates a product management system.

   The user should be able to:

   * Add products with their respective prices
   * Modify product information
   * Delete a product
   * Update product data
