# 🐍 Python Set Exercises

## 🇧🇷 Português (PT-BR)

1. Faça um algoritmo que permita ao usuário cadastrar nomes em um conjunto (set). Ao final, exiba todos os nomes informados.

2. Faça um algoritmo que receba dois conjuntos de números inteiros e exiba os principais resultados das operações entre eles.

   O algoritmo deve apresentar:

   * União dos conjuntos
   * Interseção dos conjuntos
   * Diferença entre os conjuntos
   * Diferença simétrica entre os conjuntos

---

## 🇺🇸 English (EN)

1. Write an algorithm that allows the user to register names in a set. At the end, display all the entered names.

2. Write an algorithm that receives two sets of integers and displays the main results of the operations between them.

   The algorithm must present:

   * Union of the sets
   * Intersection of the sets
   * Difference between the sets
   * Symmetric difference between the sets
