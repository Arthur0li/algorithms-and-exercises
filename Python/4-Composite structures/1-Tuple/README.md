# 🐍 Python Turple Exercises

## 🇧🇷 Português (PT-BR)

1. Faça um algoritmo que permita ao usuário digitar um número inteiro entre 0 e 20 e exiba esse número por extenso em inglês.

   Os números por extenso devem estar armazenados em uma tupla.

2. Faça um algoritmo que simule uma tabela de futebol.

   Os times devem ser armazenados em duas tuplas: uma ordenada por pontuação e outra em ordem alfabética.

   O usuário deve poder consultar:

   * Times classificados para torneios continentais
   * Times rebaixados
   * Tabela ordenada por pontos
   * Tabela em ordem alfabética

3. Faça um algoritmo que peça ao usuário a quantidade de números que deseja informar. Em seguida, armazene os valores digitados em duas tuplas: uma contendo números pares e outra contendo números ímpares.

   Ao final, exiba os números separados.

## 🇺🇸 English (EN)

1. Write an algorithm that allows the user to enter an integer between 0 and 20 and displays that number written in English.

   The numbers in written form must be stored in a tuple.

2. Write an algorithm that simulates a football league table.

   The teams must be stored in two tuples: one sorted by points and another in alphabetical order.

   The user should be able to query:

   * Teams qualified for continental tournaments
   * Relegated teams
   * Table sorted by points
   * Table in alphabetical order

3. Write an algorithm that asks the user for the number of values they wish to enter. Then, store the entered values in two tuples: one containing even numbers and the other containing odd numbers.

   At the end, display the numbers separately.
