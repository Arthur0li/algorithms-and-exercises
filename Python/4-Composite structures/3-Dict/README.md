# 🐍 Python Dict Exercises

## 🇧🇷 Português (PT-BR)

1. Faça um algoritmo que simule o registro de informações de um livro. Após coletar todos os dados, armazene-os em um dicionário e exiba as informações para o usuário.

2. Faça um algoritmo que implemente o jogo de pedra, papel e tesoura.

   As opções do jogo devem estar armazenadas em um dicionário.

3. Faça um algoritmo que receba as informações de um jogador, armazene-as em um dicionário e, em seguida, exiba esses dados para o usuário.

---

## 🇺🇸 English (EN)

1. Write an algorithm that simulates the registration of book information. After collecting all the data, store it in a dictionary and display the information to the user.

2. Write an algorithm that implements the rock, paper, scissors game.

   The game options must be stored in a dictionary.

3. Write an algorithm that receives player information, stores it in a dictionary, and then displays the data to the user.
