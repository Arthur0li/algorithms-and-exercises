# 🐍 Python Functions Exercises

## 🇧🇷 Português (PT-BR)

1. Faça um algoritmo que permita ao usuário escolher entre executar o programa em português ou inglês e, de acordo com a escolha, exiba uma saudação personalizada.

   As saudações devem estar implementadas em funções separadas.

2. Faça um algoritmo que calcule a área de diferentes figuras geométricas utilizando funções separadas para cada forma.

3. Faça um algoritmo que implemente uma calculadora simples utilizando funções separadas para cada operação matemática.

---

## 🇺🇸 English (EN)

1. Write an algorithm that allows the user to choose between running the program in Portuguese or English and, based on the choice, displays a personalized greeting.

   The greetings must be implemented using separate functions.

2. Write an algorithm that calculates the area of different geometric shapes using separate functions for each shape.

3. Write an algorithm that implements a simple calculator using separate functions for each mathematical operation.
