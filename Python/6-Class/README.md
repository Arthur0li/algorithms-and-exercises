# 🐍 Python Class Exercises

## 🇧🇷 Português (PT-BR)

1. Faça um algoritmo que represente um canal de conteúdo.

   O programa deve criar uma classe para o canal com informações como nome, descrição, quantidade de inscritos e nacionalidade.
   Também deve permitir adicionar e remover inscritos, exibir os dados do canal e criar uma classe filha para canais de negócios com uma lista de membros da equipe.

2. Faça um algoritmo que represente um computador.

   O programa deve criar uma classe com informações como marca, memória RAM e placa de vídeo.
   Também deve permitir ligar e desligar o computador, além de exibir seus dados e seu estado atual.

3. Faça um algoritmo que represente um vendedor.

   O programa deve criar uma classe para cadastrar nome, vendas e meta.
   Também deve permitir atualizar as vendas, verificar se a meta foi atingida e exibir as informações do vendedor.

4. Faça um algoritmo que represente um controle remoto de TV.

   O programa deve criar uma classe com informações como cor, altura e largura.
   Também deve permitir ligar e desligar a TV, trocar de canal, ajustar o volume, abrir o Netflix e exibir o estado atual do aparelho.

5. Faça um algoritmo que represente um cliente de uma plataforma de streaming.

   O programa deve criar uma classe com nome, e-mail e plano de assinatura.
   Também deve validar o plano, permitir a troca de plano e verificar se o usuário pode assistir a um filme de acordo com o plano contratado.

6. Faça um algoritmo que represente jogadores de futebol.

   O programa deve criar uma classe base com atributos como altura, velocidade, passe, drible e precisão.
   Em seguida, deve criar classes filhas para goleiro e jogador de linha, demonstrando herança e polimorfismo com ações como defender, passar e chutar.

7. Faça um algoritmo que represente programas de entretenimento.

   O programa deve criar uma classe base para programas, com atributos como título, descrição e trailer.
   Em seguida, deve criar classes filhas para série e filme, demonstrando herança e polimorfismo na exibição das descrições.

## 🇺🇸 English (EN)

1. Write an algorithm that represents a content channel.

   The program should create a class for the channel with information such as name, description, subscriber count, and nationality.
   It should also allow adding and removing subscribers, displaying channel data, and creating a child class for business channels with a team member list.

2. Write an algorithm that represents a computer.

   The program should create a class with information such as brand, RAM memory, and graphics card.
   It should also allow turning the computer on and off, as well as displaying its data and current state.

3. Write an algorithm that represents a seller.

   The program should create a class to register name, sales, and goal.
   It should also allow updating sales, checking whether the goal was reached, and displaying the seller's information.

4. Write an algorithm that represents a TV remote control.

   The program should create a class with information such as color, height, and width.
   It should also allow turning the TV on and off, changing channels, adjusting volume, opening Netflix, and displaying the current state of the device.

5. Write an algorithm that represents a streaming platform customer.

   The program should create a class with name, email, and subscription plan.
   It should also validate the plan, allow plan changes, and check whether the user can watch a movie according to the subscribed plan.

6. Write an algorithm that represents football players.

   The program should create a base class with attributes such as height, speed, passing, dribbling, and accuracy.
   Then, it should create child classes for goalkeeper and field player, demonstrating inheritance and polymorphism with actions such as defending, passing, and shooting.

7. Write an algorithm that represents entertainment programs.

   The program should create a base class for programs, with attributes such as title, description, and trailer.
   Then, it should create child classes for series and movies, demonstrating inheritance and polymorphism in description display behavior.
