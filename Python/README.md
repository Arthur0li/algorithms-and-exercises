# 🐍 Python Exercises

## 📖 About / Sobre

**EN:**
This directory contains Python programming exercises developed for **practice and learning**, organized by topics and concepts.

The exercises are based on **books and online courses**, mainly:

* *Python Workout: 50 Ten-Minute Exercises*
* YouTube channel *Curso em Vídeo*

The repository focuses on improving problem-solving skills, understanding Python syntax, and building consistency through practice.

**PT-BR:**
Este diretório contém exercícios de programação em Python desenvolvidos para **prática e aprendizado**, organizados por tópicos e conceitos.

Os exercícios são baseados em **livros e cursos online**, principalmente:

* *Python Workout: 50 Ten-Minute Exercises*
* Canal do YouTube *Curso em Vídeo*

O objetivo do repositório é melhorar a resolução de problemas, compreender a sintaxe do Python e desenvolver consistência através da prática.

## 🧠 Learning Approach / Abordagem de Aprendizado

**EN:**
The exercises are designed to reinforce key programming concepts, including:

* Variables and data types
* Operators (arithmetic, logical, relational)
* Conditional structures (`if`, `elif`, `else`)
* Repetition structures (`for`, `while`)
* Functions and code organization
* String manipulation
* Problem-solving and algorithm design

As the repository evolves, more advanced Python topics may be included.

**PT-BR:**
Os exercícios são voltados para reforçar conceitos fundamentais de programação, incluindo:

* Variáveis e tipos de dados
* Operadores (aritméticos, lógicos e relacionais)
* Estruturas condicionais (`if`, `elif`, `else`)
* Estruturas de repetição (`for`, `while`)
* Funções e organização de código
* Manipulação de strings
* Resolução de problemas e construção de algoritmos

Com o avanço dos estudos, tópicos mais avançados de Python podem ser incluídos.
