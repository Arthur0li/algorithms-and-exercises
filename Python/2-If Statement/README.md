# 🐍 Python If Statement Exercises

## 🇧🇷 Português (PT-BR)

1. Faça um algoritmo que analise um número e informe se ele é par ou ímpar.

2. Faça um algoritmo que converta anos humanos em anos de cachorro.

   Considere que cada um dos dois primeiros anos humanos equivale a 10,5 anos de cachorro, e cada ano adicional equivale a 4 anos de cachorro.

   O algoritmo deve funcionar corretamente para valores menores que dois anos e para valores maiores ou iguais a dois anos.

   Exiba uma mensagem de erro caso o usuário informe um número negativo.

3. Faça um algoritmo que analise uma letra digitada pelo usuário e informe se é vogal ou consoante.

4. Faça um algoritmo que simule uma roleta utilizando geração de números aleatórios.

   A roleta possui números de 1 a 36, além de 0 e 00. Os números podem ser vermelhos, pretos ou verdes.

   O algoritmo deve exibir o número sorteado e todas as apostas vencedoras, como:

   * Número
   * Cor (vermelho ou preto)
   * Par ou ímpar
   * Intervalo (1 a 18 ou 19 a 36)

   Caso o resultado seja 0 ou 00, exiba apenas o número sorteado.

5. Faça um algoritmo que receba um ano e informe se ele é bissexto ou não.

6. Faça um algoritmo que receba a quantidade de lados de uma figura e informe qual figura geométrica ela representa.

7. Faça um algoritmo que receba um ano e informe a qual signo do zodíaco chinês ele pertence.

8. Faça um algoritmo que receba os três lados de um triângulo e informe sua classificação.

9. Faça um algoritmo que simule uma calculadora simples que receba dois números e uma operação escolhida pelo usuário.

10. Faça um algoritmo que receba a quantidade de gols de uma partida e informe qual time venceu.

## 🇺🇸 English (EN)

1. Write an algorithm that analyzes a number and determines whether it is even or odd.

2. Write an algorithm that converts human years to dog years.

   Consider that each of the first two human years counts as 10.5 dog years, and each additional year counts as 4 dog years.

   The algorithm must work correctly for values less than two years and for values greater than or equal to two years.

   Display an error message if the user enters a negative number.

3. Write an algorithm that analyzes a letter entered by the user and determines whether it is a vowel or a consonant.

4. Write an algorithm that simulates a roulette spin using random number generation.

   The wheel contains numbers from 1 to 36, as well as 0 and 00. The numbers can be red, black, or green.

   The algorithm should display the selected number and all winning bets, such as:

   * Number
   * Color (red or black)
   * Odd or even
   * Range (1 to 18 or 19 to 36)

   If the result is 0 or 00, display only the number.

5. Write an algorithm that receives a year and determines whether it is a leap year.

6. Write an algorithm that receives the number of sides of a figure and determines which geometric shape it represents.

7. Write an algorithm that receives a year and determines the corresponding Chinese zodiac sign.

8. Write an algorithm that receives the three sides of a triangle and determines its classification.

9. Write an algorithm that simulates a simple calculator that receives two numbers and an operation chosen by the user.

10. Write an algorithm that receives the number of goals in a match and determines which team won.
