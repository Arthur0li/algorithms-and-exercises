# 🐍 Python For Exercises

## 🇧🇷 Português (PT-BR)

1. Faça um algoritmo que peça ao usuário para digitar um número positivo e mostre os números até chegar no número informado.

   Exemplo: se o usuário digitar 3, a saída deve ser: 0, 1, 2, 3.

2. Faça um algoritmo que peça ao usuário para digitar um número e retorne a tabela de multiplicação desse número.

3. Faça um algoritmo de soma em que o usuário informe a quantidade de números que serão somados e, em seguida, digite esses números um por um. Ao final, exiba o resultado da soma.

4. Faça um algoritmo que crie uma pirâmide numérica de acordo com o número digitado pelo usuário.

   Exemplo: se o usuário digitar 3, a saída deve ser:
   1
   1 2
   1 2 3

## 🇺🇸 English (EN)

1. Write an algorithm that asks the user to enter a positive number and displays the numbers up to the entered value.

   Example: if the user enters 3, the output should be: 0, 1, 2, 3.

2. Write an algorithm that asks the user to enter a number and returns the multiplication table for that number.

3. Write a summation algorithm in which the user enters the number of values to be added and then enters those numbers one by one. At the end, display the final sum.

4. Write an algorithm that creates a numeric pyramid according to the number entered by the user.

   Example: if the user enters 3, the output should be:
   1
   1 2
   1 2 3
